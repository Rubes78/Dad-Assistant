3 Host Shell Access (optional)

Let Fatharr run commands on your server via SSH. It uses a dedicated key pair — no passwords sent anywhere.

Copy the public key above and run this on your server:
mkdir -p ~/.ssh && echo "PUBLIC_KEY" >> ~/.ssh/authorized_keys
